给定两个长度为 $n$ 的序列 $a,b$ ，定义 $c_{i,j} = a_i+b_j$ ，输出 $c$ 的 $n\times n$ 个元素中前 $k$ 小

$n\le10^5$

<!-- more -->

---

核心代码

```cpp
struct S {
    int i, j;
    bool operator<(const S &rhs) const { return a[i] + b[j] > a[rhs.i] + b[rhs.j]; }
};
std::priority_queue<S> q;
rep(i, 1, n) q.push({i, 1}); // c[i][1]
rep(cnt, 1, n) {
    // pop c[i][j], push c[i][j+1]
    int i = q.top().i, j = q.top().j;
    q.pop();
    io.write(a[i] + b[j], ' ');
    ++j;
    if (j <= n) q.push({i, j});
}
```
