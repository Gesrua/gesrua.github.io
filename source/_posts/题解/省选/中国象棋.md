给定 $n\times m$ 棋盘，每格只能放 $1$ 个棋子，每行每列最多放 $2$ 个棋子，问方案数量

<!-- more -->

---

我好菜啊，又不会做

$f_{i, j, k, l}$ 表示当前处理到第 $i$ 行，有 $j$ 列放的数量为 $1$ ，有 $k$ 列放的数量为 $2$ ，有 $l$ 列放的数量为 $0$

考虑 $j+k+l = m$ 可以压掉一维

转移

```cpp
// C(n) = C(n, 2) = n * (n - 1) / 2;

// 0
f[i + 1][j][k] += f[i][j][k];

// 1
// l->j
if (l >= 1) (f[i + 1][j + 1][k] += f[i][j][k] * l) %= p;
// j->k
if (j >= 1) (f[i + 1][j - 1][k + 1] += f[i][j][k] * j) %= p;

// 2
// l->j l->j
if (l >= 2) (f[i + 1][j + 2][k] += f[i][j][k] * C(l)) %= p;
// l->j j->k
if (l >= 1 && j >= 1)
    (f[i + 1][j][k + 1] += f[i][j][k] * l * j) %= p;
// j->k j->k
if (j >= 2) (f[i + 1][j - 2][k + 2] += f[i][j][k] * C(j)) %= p;
```
