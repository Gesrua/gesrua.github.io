更优雅的线段树

线段树原理非常简单，但如何优雅的实现却是一个问题

¶ 线段树简介

线段树就是把一个序列原有 nnn 个点扩充为 nlog⁡nn\log nnlogn 个点，保持一定结构进行加速

每个点有

  • 自己管辖的区域
  • 两个拼起来是自己的子节点（ self=lson∪rsonself = lson \cup rsonself=lson∪rson ）
  • 子节点不交（ lson∩rson=∅lson \cap rson = \varnothinglson∩rson=∅ ）
  • 可以快速的区间合并
